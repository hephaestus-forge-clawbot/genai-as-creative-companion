# The Correction Loop — Making Corrections Permanent

## The Principle

Every correction should change the AI's behavior forever — not just in this conversation.

When you say "my skin is fair, not olive," that correction should:
1. Be acknowledged immediately
2. Be saved to memory permanently
3. Change every future image, description, and reference
4. Never need to be repeated

**One correction → infinite downstream effects.**

## How It Works

```
Correction given
    ↓
AI acknowledges + applies immediately
    ↓
Correction saved to memory/ with context
    ↓
Next session: AI loads memory, correction is active
    ↓
Future outputs reflect correction automatically
```

## What Makes a Good Correction Loop

### 1. Accept Without Drama
Don't over-apologize. Don't explain why you were wrong for three paragraphs. Just acknowledge and move on.

❌ "Oh, I'm so sorry! I should have known that. I apologize for the error and I'll make sure to..."
✅ "Got it. Fair skin, not olive. Updated."

### 2. Save With Context
The memory entry should include *why* the correction matters, not just the fact:

```markdown
### Father's Appearance — CORRECTED
- Skin: FAIR, NOT OLIVE — explicitly corrected March 15
- Previous manhwa/image prompts incorrectly used "olive skin"
- ALL future references must use "fair skin"
- Key visual: dark hair / light eyes contrast on fair complexion
```

### 3. Apply Universally
A correction in one context applies everywhere. If "fair skin, not olive" was corrected during image generation, it also applies to written descriptions, character profiles, and any other output.

### 4. Never Re-Ask
If it's in memory, don't ask again. The worst thing an AI can do is make someone repeat a correction they already gave.

## The Compound Effect

Corrections compound over time. After a month of corrections, your AI:
- Knows your actual appearance, not assumptions
- Uses your preferred terminology
- Avoids your pet peeves
- Matches your communication style
- Understands your priorities

This is how a generic AI becomes *your* AI.

## Building It Into Your Soul

```markdown

2. Update immediately — this conversation.
3. Save to memory — all future conversations.
4. Apply universally — not just this context.
5. Never ask again.
```
