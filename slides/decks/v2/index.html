GenAI as Creative Companion — v2 Designs

5 radically different design languages. Same 15-slide script. Arrow keys to navigate, F for fullscreen, N for speaker notes.

1. Apple Keynote
White/clean, premium minimalism. Big photography, bold typography.
2. Terminal / Hacker
Dark console, monospace, green-on-black. Git diff as memory metaphor. The audience is reading logs.
3. Magazine Editorial
Serif typography, two-column layouts, pull-quotes, drop caps. Reads like a New Yorker feature.
4. Neon Cyberpunk
Deep black with cyan/magenta neon. Glassmorphism panels, Orbitron titles, synthwave vibes.
5. Gallery / Museum
Art exhibition walk-through. Dark gallery walls, Roman numeral rooms, curated exhibit labels.
