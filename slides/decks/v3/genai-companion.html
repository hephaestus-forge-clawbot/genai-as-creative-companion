The Model: Claude Opus 4

Everything you just saw was generated with Anthropic's Claude Opus (4.5 → 4.6).

Why This Model?

  • Doesn't degenerate on long tasks — most LLMs lose coherence after ~10K tokens. Opus maintains quality across a 14,500-word novella.
  • Self-heals — catches inconsistencies, corrects tone drift, maintains character voice across thousands of lines.
  • Massive context window — can hold the entire project state (novella + character profiles + world-building) in a single session.
  • Genuine creative range — not just competent prose. Emotionally resonant fiction that real people recognised themselves in.
The model is necessary but not sufficient. You also need a system that knows how to use it.

The Framework: OpenClaw

OpenClaw — open-source framework for persistent AI agents.

  • 🧠 Memory — agents remember across sessions, not just within a chat
  • 🛠️ Tools — file system, web, code execution, browser, APIs — all pluggable
  • 👤 Personality — configurable identity via SOUL.md and AGENTS.md
  • 🔄 Persistent state — agents maintain context, goals, and project knowledge
  • 📦 Skills — modular capability packs (image gen, music, research, coding)

github.com/openclawai/openclaw

Not a chatbot. A system that maintains state, executes multi-step creative workflows, and grows with you.

The Soul: Hephaestus

Hephaestus — the personality/soul system built on OpenClaw. Named after the Greek god of the forge.

Key Ingredients

  • SOUL.md — defines identity, anti-sycophancy rules, genuine creative voice
  • AGENTS.md — methodology, Byzantine verification, quality gates
  • Memory — persistent knowledge across weeks of interaction
  • Skills — modular capabilities (image gen, research, code, music)

Byzantine Verification

Nothing ships without independent verification. Multiple agents check each other's work:

  • 2-1 minimum for content extraction
  • 3-2-1 pyramid for production builds
  • 5-3-2-1 pyramid for specifications
